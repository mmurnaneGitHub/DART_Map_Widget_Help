Identify - Identify map layer details. 

Click anywhere on the map to query over thirty datasets with results appearing in 'Identify Results' panel. Blue text without underlines may be clicked to show data boundary. Blue text with underlines are links to additional information (websites, documents).

Items within the appropriate scale range will appear in the Identify Results popup regardless if they are turned on in the legend. If the desired items are not appearing in the popup, try zooming in first.

Additional information on any layer may be obtained by using the Enable Pop-up option in the Layer List.

 

  Basemap Gallery - Change background map. 

Select a background basemap that includes standard maps, aerial photography, muted, and more. Individual layers within a basemap cannot be turned off.

More Details

 

  Legend - View visible map layers.

Shows all layers currently visible on the map. Layers must be turned on and the map within their visible scale range to appear in the legend.

More Details

 

  Layer List - Toggle visibility & transparency.

Looking for a layer? Click the search button on upper right of panel and type a keyword to find specific layers.

Each layer and groups of layers have a checkbox for visibility. Layers that are gray are not visible at the current map scale.

Each item in the list is either a map layer or group of layers and has an expand/contract arrow followed by a checkbox for visibility, the layer or group name and then '...' to access a mini-menu.

The mini menu gives you control of the layer or group transparency and also lets you move it up or down in the list. The order that items are listed is also the order they are drawn on the map. If the item is a group of layers, clicking the expand arrow will open the group to show all of the map layers it contains. Each layer will have its own expand arrow, visibility checkbox and mini-menu. The layer's expand control reveals the layer's map symbology, while the mini-menu allows you to open up the layer's attributes in the "attribute table". You cannot change the order of layers within the group.

From the mini-menu Enable Pop-up / Remove Pop-up enables or disables the pop-up for the feature layer. To see the feature layer pop-up, close the 'Identify Results' window. If a feature layer does not have a pop-up configured in the map, clicking Enable Pop-up shows all field values for the feature layer.

More Details

 

  Add Data List- Add additional data to map. 

Additional data may be added and removed from the map using a checkbox menu.

 

  Add ESRI Data - Add AGOL data to map. 

Add ESRI ArcGIS Online data (over 100k layers) to the map. Layers may also be added using a web service URL or dragging a file (shapefile, csv - matched with generic ESRI geocoder, gpx, geojson) onto the panel tab.

More Details

 

  Add CSV File - Add a CSV file to map.

Drag and drop a CSV file (comma separated values, create using 'Save as ...' in Excel) onto panel to automatically map points. File must have a field named Address (no city, state, zip values) OR fields named Latitude and Longitude. Field names must contain the actual upper and lower case shown in the examples. Address will be matched with Tacoma address points. Close the 'Identify Results' panel and click the resulting map dots to view the spreadsheet values. If errors occur on any row of the spreadsheet, details are provided on the panel.

 

  Map Themes - Swap map layers & background.

Swap map layers and backgrounds grouped for research including permitting, bus transportation, utilities, and equity. Current map extent will be preserved. Layers added to map will be removed, so switch map themes first and then add layers.

 

  Measure - Calculate length and area.

Add lines and polygons to map with length and area labels. Options include labels for measurement totals and bearing.

More Details

 

  Street View - See Google® Street View.

See Google Street View images for map location. Activate the tool, select from three Street View options, and then click the map. Google Street View will open in a new window, so be sure popup blocker allows site.

IE browsers may experience a blank screen. To Fix:

  1. Open up Internet Explorer
  2. Press the Alt key on keyboard (menu bar appears)
  3. Click on the Tools menu tab
  4. Select the "Compatibility View settings" option
  5. Uncheck the "Display intranet sites in Compatibility View" option

If a Chrome browser does not load the Google Earth option, go to advanced settings and turn on `Use hardware acceleration when available'.

  Draw - Add and save markups.

Add, edit, save, import, export, and share map markups.

Exported markups (json file) can be converted to data (feature class) using the ArcTool JSON To Features (Conversion). The resulting feature class will have fields, geometry type, and spatial reference as defined in the JSON.

 

  Print - Print current map.

Create a map to print or save. Change title, layout size, and format type. Use the 'Advanced' button for scale, metadata, map only size, and print quality options. If basemap is not displaying in resulting map, choose "Preserve map extent" under advanced options and print again.

More Details

 

  Favorites - Save current map settings.

Save extent and layer visibility of current map session. Retrieve and share previously saved sessions to start map with the same settings. Session files are stored in the browser cache by default. Store favorites at a network location to prevent accidental deletion. The current basemap is not saved in the favorite, only the layers in the Layer List.

NOTE: Don't not create favorites from existing favorites. If a favorite is currently in use, refresh browser with Ctrl F5, and then compose the new favorite.

Workflow for sharing favorites:

  1. Create favorites with Chrome or FireFox, then select the `Save to File' option to create a json text file and save to a location outside the browser cache folder.
  2. Using any browser, including IE and Edge, select the `Load to file' option to use the saved json text file.

 

  Search - Find address, parcel, or place name.

Search for a location by address, intersection, parcel number, or place name. Use the & symbol to separate street names on an intersection search (ie 9th & Market). The same search term can be added to the URL with the '?find' prefix to create automatic links to a location (ie https://wspdsmap.cityoftacoma.org/website/DART/staff/TacomaPermitsMap.htm?find=1515 Commerce St).

More Details

 

  Default extent - Zoom map to all of Tacoma.

Use this button to return the map to all of Tacoma.

More Details

 

  Find my location - Zoom map to current location.

Map will zoom to your current location using the browser's geolocation feature. The map location will be fairly accurate on mobile devices, but only a gross estimate on most desktop devices. However, desktop devices can be made highly accurate in the Microsoft Edge browser by adjusting your Location within Windows 10 Settings.

More Details

 

      Get coordinates - Determine latitude & longitude.

Located on lower left side of map. Shows the current map coordinates of the mouse location. Click left side target icon and then click on map to capture coordinates for copying elsewhere (ie spreadsheet).

More Details

 

         Open Attribute Table - View map layer tables.

Located at bottom center of map. Attribute table may also be opened from Layer List data mini-menu. Displays a tabular view of the layers' attributes. Includes options for filtering and exporting selected records.

More Details

 

MAP NAVIGATION:

  • CENTER & ZOOM IN - Double Click
  • PAN - Drag the mouse with button pressed
  • RECENTER - Shift + Click
  • ZOOM IN - Mouse scroll forward
  • ZOOM IN (rubber band) - Shift + Drag the mouse
  • ZOOM OUT - Mouse scroll backward
  • ZOOM OUT (rubber band) - Shift + Ctrl + Drag
